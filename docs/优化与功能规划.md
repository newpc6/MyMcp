# 优化与功能规划

本文档记录当前项目后续可以继续优化和新增的方向，便于拆分任务、评估优先级和持续迭代。

## P0：稳定性与规范收敛

### 1. 后端 Repository 补齐

当前 `repositories/` 主要覆盖用户和租户，MCP 模板、发布服务、统计等模块仍有部分查询逻辑分散在 model/service 中。建议逐步补齐：

- `mcp_template_repository.py`
- `published_service_repository.py`
- `mcp_template_group_repository.py`
- `statistics_repository.py`
- `mcp_auth_repository.py`

目标是让 API 和 Service 不直接拼复杂 SQL，新增功能默认走 Repository。

### 2. 数据迁移版本化

已有启动时迁移能力，但后续可进一步增强：

- 记录已执行迁移版本，避免只依赖脚本幂等判断。
- 启动时输出迁移摘要，便于部署排障。
- 提供 `--check-migrations` 只检查不执行的命令。
- 为 SQLite/MySQL 差异建立统一 helper。

### 3. 前端类型收敛

当前仍有部分 `any` 和内联类型，建议逐步处理：

- 将模板配置参数、工具参数、执行结果定义为明确类型。
- 将表格分页响应抽成统一 `PageResult<T>`。
- 将错误响应抽成统一 `ApiError`。
- 移除页面里的重复分页状态，沉淀 `useTable`。

## P1：用户体验优化

### 1. 全局搜索

在顶部提供全局搜索入口，支持快速搜索：

- MCP 模板
- 已发布服务
- 工具函数
- 密钥
- 访问日志

搜索结果可按模块分组展示，点击后跳转对应页面。

### 2. 模板详情体验

模板详情页可继续增强：

- README/Markdown 目录导航。
- 工具函数列表支持按函数名、参数、描述过滤。
- 工具测试历史记录。
- 配置参数示例值和校验提示。
- 发布前配置检查清单。

### 3. 服务运维能力

服务管理页可以增加：

- 批量启动/停止服务。
- 服务健康检查和最近错误摘要。
- 服务运行时配置差异对比。
- 服务复制发布。
- 第三方服务连通性定时检测。

### 4. 访问日志分析

访问日志页可以增加：

- 错误率趋势。
- 高耗时请求排行。
- 按密钥、服务、状态码聚合。
- 日志导出 CSV。
- 可配置保留周期和清理任务。

## P2：平台能力扩展

### 1. 模板版本管理

为 MCP 模板增加版本能力：

- 模板版本号和变更说明。
- 版本发布/回滚。
- 服务绑定模板版本。
- 不同版本工具列表对比。

### 2. 模板导入导出

支持模板包能力：

- 导出模板元数据、代码、参数 schema 和 README。
- 导入模板包并进行冲突检查。
- 支持模板市场离线迁移。

### 3. 权限细化

当前权限可以继续细化：

- 模板查看/编辑/发布权限。
- 服务启动/停止/删除权限。
- 密钥查看明文/重置权限。
- 访问日志查看范围。

### 4. 审计增强

统一记录关键操作：

- 模板创建、编辑、删除、复制。
- 服务发布、启动、停止、卸载。
- 密钥创建、禁用、删除。
- 用户和权限变更。

审计日志应支持筛选、导出和详情查看。

## P3：工程质量提升

### 1. 前端公共组件沉淀

建议补齐以下组件或组合式函数：

- `PageContainer`
- `PageHeader`
- `DataTable`
- `SearchToolbar`
- `StatusDot`
- `useTable`
- `useConfirmAction`

减少页面内重复布局和交互代码。

### 2. 样式债务清理

当前部分历史页面仍有硬编码颜色、强渐变和重复阴影。建议按页面分批处理：

- 统计分析页。
- 用户管理页。
- 租户管理页。
- 模板工具测试页。
- 服务发布卡片。

每次只处理一个模块，避免大范围视觉回归。

### 3. 自动化验证

建议增加：

- 后端基础 import/compile 检查脚本。
- 前端 build 和 lint 检查脚本。
- 关键页面 Playwright 冒烟测试。
- Markdown 链接和图片大小检查。

## 建议执行顺序

1. 补齐项目级公共组件和 `useTable`，先降低前端重复代码。
2. 给发布服务和模板模块补 Repository，收敛后端数据库访问。
3. 做模板详情和工具测试体验优化。
4. 做服务健康检查、日志分析和审计增强。
5. 做模板版本管理和导入导出。
